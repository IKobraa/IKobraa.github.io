Tagesschau

Die Tagesschau ist eine deutsche Nachrichtenagentur

Idea

for our Big Data Project we chose to gather Information from the News Website Tagesschau.de and then analyze, visualize and interpret the gathered information. Data that we gathered are headlines,short description,date,time,author,url,teaser and the full article text.

Architecture

We chose to write a Webcrawler in python because of the how effortless one can write a Webcrawler with it.
